Keyboard Shortcuts

Click canvas — create node
Click node — select (then click another to connect)
Double-click node — edit text
Right-click node — cycle color (idea/question/viable/non-viable)
Delete / Backspace — remove selected
Ctrl+Z — undo
Ctrl+Shift+Z — redo
Escape — cancel / deselect
Scroll — zoom in/out
Drag canvas — pan
Drag node — move
Click connector — edit label
H — toggle this HUD
